ShExStatements
==============

ShExStatements converts CSV-like statements and spreadsheet files into Shape Expressions (ShEx).

The documentation below is sourced from Markdown files in this repository to avoid duplicate copies.

Supported usage modes
---------------------

- WASM runtime in the browser (no backend required)
- Docker runtime (React frontend + FastAPI backend)
- Python runtime (CLI and legacy Flask interface)
