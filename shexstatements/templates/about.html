About ShExStatements

Objectives

  • Easily generate shape expressions (ShEx) from CSV and Spreadsheet files
  • Simple 5-column syntax

Source code

  • ShExStatements on Github

Conferences

Demonstration

Author

  • John Samuel
  • Contributors

Technologies

ShExStatements can be tested on any modern browser and on the command line. Following are the technologies used for creating ShExStatements:

  • HTML, Javascript, CSS, Python
  • Python libraries: flask, ply, pyshex, openpyxl, xlrd, odfpy
